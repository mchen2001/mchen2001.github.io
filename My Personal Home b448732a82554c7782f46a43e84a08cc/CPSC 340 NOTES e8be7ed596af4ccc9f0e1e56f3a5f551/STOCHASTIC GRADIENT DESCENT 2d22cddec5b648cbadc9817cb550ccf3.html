STOCHASTIC GRADIENT DESCENT

Gradient descent is effective when 'd' is very large. But if n is large, it will take more time. Therefore, stochastic gradient is introduced to solve this problem.

Why Stochastic Gradient Descent Will Work?

The expected direction is the full gradient.

The algorithm is going in the right direction on average.

Effect of the Step-Size

var∝stepSize2var \propto stepSize^2var∝stepSize2﻿

  • Variance decreases as step size decreases.
  • For a fixed step-size, SG makes progress until variance is too big.

This leads to two phases:

  1. Rapid progress when we are far from the solution.
  1. Erratic behaviour confined to a “ball” around solution.

Stochastic Gradient with Decreasing Step Sizes

To get convergence, we need to decrease the step size

Stochastic Gradient with Constant Step Sizes

  • This will not converge
  • But there is a step-size small enough to achieve a fixed accuracy

You can use a bigger step size as the batch size increases: Gets you to the ball faster (though diverges if step-size gets too big).

A Practical Strategy for Deciding When to Stop

  1. Every ‘k’ iterations (for some large ‘k’), measure validation set error
  1. Stop if the validation set error “isn’t improving”: this can also reduce overfitting

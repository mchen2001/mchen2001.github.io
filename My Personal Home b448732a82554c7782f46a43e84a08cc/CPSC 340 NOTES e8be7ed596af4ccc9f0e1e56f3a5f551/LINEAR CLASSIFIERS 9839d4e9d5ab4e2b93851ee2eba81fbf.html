  • It is convex: max{constant,linear}max\{constant, linear\}max{constant,linear}﻿
  • it is not degenerate as w=0w=0w=0﻿ now gives error of δ\deltaδ﻿ instead of 000﻿

Hinge Loss for all 'n' training examples is given by: f(w)=∑i=1nmax{0,δ−yiwTxi}f(w)=\sum_{i=1}^n max\{0,\delta-y_iw^Tx_i\}f(w)=∑i=1n​max{0,δ−yi​wTxi​}﻿

Convex upper bound on 0-1 loss:

  • If the Hinge Loss is 18.3 then number of training error is at most 18
  • So minimizing hinge loss indirectly tries to minimize training error
  • Like perceptron, finds a perfect linear classifier if one exists
